# Project 01 — Digital Banking App Revamp
### Domain: Retail Banking | Role: Business Analyst

---

## Overview
A private sector bank's mobile banking application had a dated UI, 
no UPI integration, and poor navigation — resulting in 35% of all 
support tickets being login or payment related. This project covers 
the complete BA documentation for a full mobile app revamp targeting 
a 30% increase in digital transaction volume.

---

## Business Objective
- Increase digital transaction volume by 30%
- Reduce login-related support tickets by 25%
- Improve app store rating from 3.2 to 4.2 stars
- Integrate UPI payments per RBI guidelines

---

## Scope
- Mobile app UI redesign (iOS 14+ and Android 10+)
- Biometric login — Face ID and Fingerprint
- UPI and IMPS real-time payment integration
- AI-driven spending analytics dashboard
- Push notification system for all transactions

---

## Epics Covered
| Epic | User Stories |
|------|-------------|
| Biometric Authentication | BA-001, BA-002 |
| UPI Integration | BA-003 |
| Spending Analytics | BA-004 |

---

## Key Business Rules
- Transactions above Rs.50,000 trigger 2-Factor Authentication
- UPI daily limit capped at Rs.1,00,000 per RBI guidelines
- Session timeout after 5 minutes of inactivity
- Biometric data stored on-device only — never on server

---

## UAT Coverage
8 test cases covering biometric login, session timeout, 
UPI transfers, daily limit enforcement, and analytics rendering.

---

## Compliance
RBI Digital Banking Guidelines 2023
